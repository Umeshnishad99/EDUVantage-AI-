# 🎓 EduVantage AI - Student Performance Prediction Platform

**EduVantage AI** is a premium, AI-driven educational analytics platform designed to bridge the gap between static academic data and actionable student success strategies. Using advanced Machine Learning and Generative AI, it provides students with personalized roadmaps and educators with deep institutional insights.

---

## 🚀 Key Features

- **🔮 Predictive GPA Modeling**: High-precision GPA forecasting using Random Forest algorithms.
- **🗺️ Personalized Roadmaps**: Dynamic, AI-generated career and academic roadmaps based on current performance tiers.
- **🤖 Gemini-Powered Chatbot**: 24/7 AI guidance for students to resolve academic queries and get real-time mentorship.
- **📊 Institutional Overview**: A high-fidelity dashboard for teachers with Recharts-powered analytics for tracking student distribution and alerts.
- **🎨 Premium UI/UX**: Modern, responsive design with dark mode support, glassmorphism, and smooth animations using Framer Motion.

---

## 🛠️ Tech Stack

### Frontend
- **React 19** with **TypeScript**
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling)
- **Lucide React** (Icons)
- **Framer Motion** (Animations)
- **Recharts** (Data Visualization)

### Backend
- **Node.js** & **Express**
- **PostgreSQL** (Database)
- **Google Gemini AI SDK** (Generative AI Integration)

### Machine Learning Service
- **FastAPI** (Python)
- **Scikit-learn** (Prediction Models)
- **Pandas & NumPy** (Data Processing)

## ⚙️ Installation & Setup

### 1. Prerequisites
- Node.js (v18+)
- Python (v3.10+)
- PostgreSQL

### 2. Backend Setup
```bash
cd backend/src
npm install
# Create a .env file and add your DATABASE_URL, PORT (5000), and GEMINI_API_KEY
npm run dev
```

### 3. ML API Setup
```bash
cd ml-api
pip install -r requirements.txt
python main.py
```

### 4. Frontend Setup
```bash
cd frontend/frontend
npm install
npm run dev
```
